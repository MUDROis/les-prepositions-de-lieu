# 🐱 Où est le chat? — Французские предлоги места

**Интерактивная игра для изучения французских предлогов: SUR, SOUS, DANS**

Веселый и наглядный способ для детей и новичков освоить три самых распространённых французских предлога места через интерактивные задания и аудиоподдержку.

---

## 🎯 Что учит эта игра

Эта интерактивная викторина помогает учащимся освоить три основных французских предлога места:

| Французский | Английский | Пример на французском | Перевод |
|-------------|------------|----------------------|---------|
| **SUR** | на (поверхность) | sur la table | на столе |
| **SOUS** | под | sous le lit | под кроватью |
| **DANS** | в (внутри) | dans la boîte | в коробке |

**Уровень:** A1 (Начинающий)  
**Тема:** "Где котик?" — играемый контекст, который удерживает внимание учащихся

---

## 📋 Как использовать эту игру

### Для преподавателей

#### 🎓 Интеграция в план урока

1. **Разминка (5–7 минут)**
   - Покажите шпаргалку с предлогами и примерами
   - Воспроизведите аудио-кнопки для демонстрации правильного произношения
   - Попросите учащихся проговорить фразы вслед за вами

2. **Закрепление (10–12 минут)**
   - Покажите игру на экране/доске
   - По очереди приглашайте учащихся выбирать ответы
   - После каждого ответа обсудите, почему он верный/неверный
   - Поощряйте учащихся объяснять свои рассуждения по-французски

3. **Самостоятельная работа (5–7 минут)**
   - Учащиеся играют индивидуально на планшетах/компьютерах
   - Поощряйте их использовать аудио-функцию для тренировки произношения

4. **Обзор и расширение (5 минут)**
   - Просмотрите итоговую таблицу
   - Предложите учащимся составить 1–2 предложения, используя каждый предлог
   - Опционально: нарисовать сцену и описать её с помощью предлогов

#### 💡 Методические советы

- **Акцент на визуальном контексте**: Игра использует чёткие визуальные подсказки (положение котика) для закрепления значения
- **Активное слушание**: Всегда включайте аудио перед тем, как учащиеся выберут ответ, для развития навыков восприятия на слух
- **Взаимокоррекция**: После показа объяснений попросите учащихся объяснить исправление своими словами
- **Расширение лексики**: Используйте игру как трамплин для введения родственных слов (lit, table, boîte, couverture)

#### 📊 Идеи для оценки

- **Формирующая**: Наблюдайте за учащимися во время закрепления; фиксируйте типичные ошибки
- **Самооценка**: Попросите учащихся просмотреть итоговую таблицу и определить, какие предлоги им показались наиболее сложными
- **Творческое задание**: После игры попросите учащихся составить свои собственные вопросы "Где котик?" с новыми сценами

---

### Для родителей

#### 🏠 Использование игры дома

1. **Создайте рутину**
   - Играйте вместе 10–15 минут, 3–4 раза в неделю
   - Превратите это в весёлое "приключение с котиком", а не в тест

2. **Будьте партнёром в обучении**
   - Задавайте вопросы типа: "А где, по-твоему, находится котик?"
   - Поощряйте полные фразы по-французски: "Le chat est SOUS le lit"
   - Используйте кнопку-громкоговоритель вместе: слушайте, затем повторяйте

3. **Празднуйте прогресс**
   - Хвалите за усилия, а не только за правильные ответы
   - В конце просматривайте итоговую таблицу вместе
   - Отмечайте улучшения во времени

#### 💬 Темы для разговора

После игры задайте ребёнку:
- "Quel est ton préféré? SUR, SOUS ou DANS?" (Какой твой любимый? НА, ПОД или В?)
- "Peux-tu me montrer SOUS la table?" (Покажи мне ПОД столом?)
- "Où est ton jouet?" (Где твоя игрушка?) — затем попросите ответить с помощью предлога

#### 🎨 Дополнительные задания

- **Рисуй и описывай**: Пусть ребёнок нарисует комнату с игрушками и опишет их положение, используя три предлога
- **Поиск предлогов**: Найдите предметы дома и опишите их положение ("La chaise est SUR le tapis")
- **Сказочное путешествие**: Придумайте короткую историю о "Le chat perdu" (потерянном котике), используя предлоги

## 🎮 Особенности игры

### 🎨 Визуальное обучение
- Каждое задание показывает чёткий визуальный контекст (изображение + эмодзи)
- Цветовая кодировка ответов: зелёный — верно, красный — неверно
- Полоса прогресса показывает статус выполнения

### 🔊 Аудиоподдержка
- Кнопка-громкоговоритель 🔊 рядом со всеми фразами
- Французское произношение демонстрируется голосом носителя языка
- Поддерживает развитие навыков аудирования и говорения

### 💬 Чёткие объяснения
- После каждого ответа учащиеся видят, почему он верный/неверный
- Объяснения включают как французский, так и английский/русский компоненты

### 📊 Отслеживание прогресса
- Полоса прогресса в реальном времени: "Répondu: X/6"
- Итоговая таблица показывает все ответы с корректировками

## 🐛 Устранение неполадок

**Звук не работает?**
- Проверьте разрешения браузера на звук
- Попробуйте другой браузер (рекомендуется Chrome или Edge)
- Некоторые браузеры требуют взаимодействия пользователя перед воспроизведением звука

**Изображения не загружаются?**
- Убедитесь, что файлы изображений (1.png, 2.png и т.д.) находятся в той же папке, что и index.html
